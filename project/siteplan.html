Site Plan: Professional Developer Portfolio

Developer: Farai Rwambiwa

1. Site Name

Name: Farai Rwambiwa | Developer Portfolio

Reason: Using my own name serves as the primary identifier for personal branding. It aligns perfectly with my existing domain (farairwambiwa.com) and clearly tells recruiters and hiring managers exactly whose work they are reviewing.

2. Site Purpose

The purpose of this website is to serve as an interactive, digital resume and project showcase. It will dynamically display my technical skills (such as Python, C#, and SQL), my professional certifications, and my completed coding projects. It is designed to demonstrate frontend architecture capabilities to prospective employers while providing a direct way to contact me for remote software development internships.

3. Scenarios

  • Scenario 1: "What specific programming languages and database technologies has this developer used in practical, deployed projects?"
  • Scenario 2: "How can I view the source code for the AI generation tools and C# applications this developer has built, and how do I contact them for an interview?"

4. Color Schema

A modern, "dark mode" developer aesthetic will be used to reduce eye strain and make the code/projects pop.

  • Dark Slate (#1E1E24): This will be used as the primary background color for the body and main sections of the website.
  • Off-White (#F8F9FA): This will be used for all primary paragraph text, list items, and standard reading text to ensure high contrast and accessibility against the dark background.
  • Electric Blue (#00B4D8): This will be used as an accent color for headings (h1, h2, h3), hyperlinks, and button hover states to draw attention to key interactive elements.

5. Typography

  • Fira Code (Monospace): Used for the site header, navigation links, and all headings. This provides a technical, IDE-like "coding" vibe that appeals to tech recruiters.
  • Open Sans (Sans-serif): Used for all body text, paragraphs, and lists. It is highly legible and clean, ensuring the project descriptions and resume details are easy to read.
